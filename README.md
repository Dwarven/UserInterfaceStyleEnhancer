A utility that enhances and manages handling of UIUserInterfaceStyle.

# Podfile
To integrate UserInterfaceStyleEnhancer into your Xcode project using CocoaPods, specify it in your `Podfile`:

```ruby
pod 'UserInterfaceStyleEnhancer'
```
